Esta pasta apresenta os arquivos relacionados a implementação da solução do Desafio 1.
A solução adotada foi a utilização de uma estrutura tipo Fila, FIFO (primeiro a entrar é o primeiro a sair), desta forma as primeiras requisições sempre terão prioridade ao serem removidas da fila.

Alterações foram realizadas no arquivo original teste.c para se adequar às funcionalidades desenvolvidas no código apresentado, no entanto as alterações não comprometem o comportamento esperado da estrutura utilizada.
